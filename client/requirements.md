## Packages
rtl-css-js | For handling RTL specific CSS logic if needed
lucide-react | Icons for the UI
date-fns-jalali | For Persian date formatting
recharts | For dashboard charts (building stats)
framer-motion | For smooth page transitions and animations
react-hook-form | Form handling
zod | Schema validation
@hookform/resolvers | Zod resolver for react-hook-form

## Notes
- The application uses RTL layout.
- Authentication is handled via a custom `useAuth` hook interacting with `/api/auth/login`.
- Chat is polling-based.
- Font: Vazirmatn (via Google Fonts) for Persian language support.
